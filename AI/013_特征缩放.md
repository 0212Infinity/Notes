## 数据归一化
```shell
数据归一化就是把数据的取值范围处理为0-1或者-1-1之间

任意数据转化为0-1之间：
newValue = (oldValue-min)/(max-min)

任意数据转化为-1-1之间：
newValue = ((oldValue-min)/(max-min)-0.5)*2
```

## 均值标准化

```shell
均值标准化

x为特征数据，u为数据的平均值，s为数据的方差
newValue = (oldValue-u)/s

例子: 
(1,3,5,7,9)
u = (1+3+5+7+9)/5=5
s = ((1-5)^2+(3-5)^2+(5-5)^2+(7-5)^2+(9-5)^2)/5=8
(1-5)/8=-1/2
(3-5)/8=-1/4
(5-5)/8=0
(7-5)/8=1/4
(9-5)/8=1/2
```
